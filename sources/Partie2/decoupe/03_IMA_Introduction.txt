IMA : BOEING B777
• Caractéristiques générales :
• Débit de 2 à 8 Mbits/s (paire torsadée)
• Structure de la trame :

• Un message est composé d’au plus 31 mots (word string)
• Un mot est composé d’un label et jusqu’à 256 mots de données
• Un Label se compose de :
• Sync: 3 bits pour la synchronisation
• Cid (Channel Identification): identifie l’émetteur quand il y a plusieurs équipements identiques
(redondance)
• P : Bit de parité

• Deux protocoles incompatibles d’accès au bus :
• Basic Protocol (BP) et Combined Protocol (CP)

67

IMA : BOEING B777
• Basic Protocol (BP) :
• TGi (Terminal Gap) : Temps d’attente du terminal « i » après la dernière activité sur le bus
avant d’émettre;
• TI (Transmit Interval) : Temps d’attente pour chaque terminal entre deux de ses
transmissions consécutives;
• SG (Synchronization Gap) : Utilisé pour synchroniser tous les terminaux au début de
chaque nouveau cycle de communication;
• Mode de transmission périodique pour des grands TI

• Mode de transmission apériodique pour des petits TI

68

IMA : BOEING B777
• Combined Protocol (CP) :
• Envoi combiné de données périodiques et apériodiques
• Level 1 (L1): Phase de transmissions periodiques
• Level 2 (L2): Phase de transmission de petits (plus haute priorité) messages apériodiques
• Level 3 (L3): Phase de transmission de longs (plus basse priorité) messages apériodiques
• TI uniquement pour les messages périodiques
=>CE (Concatenation event) permet de supprimer tous les TI non terminés
• PSG (Periodic Synchronization Gap) : temps d’attente pour la synchronisation avant chaque nouveau
cycle
• ASG (Synchronization Gap) : Temps d’attente entre les phases
• AT (Aperiodic Time-out) : Pour interrompre la diffusion des messages apériodiques et éviter un
débordement du cycle.

• Pour chaque cycle transmission de messages de level 1 et 2.
• Les messages du level 3 peuvent être retardés
